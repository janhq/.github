# .github

This is the default community health files repository for the [Jan](https://github.com/janhq) organization.

## What's Included

| File | Purpose |
|------|---------|
| [profile/README.md](profile/README.md) | Organization profile displayed on GitHub |
| [assets/](assets/) | Logo assets for the profile |
| [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) | Community code of conduct |
| [CONTRIBUTING.md](CONTRIBUTING.md) | Contribution guidelines |
| [SECURITY.md](SECURITY.md) | Security policy |
| [SUPPORT.md](SUPPORT.md) | Support resources |
| [.github/ISSUE_TEMPLATE/](.github/ISSUE_TEMPLATE/) | Issue templates (bug report, feature request) |
| [.github/pull_request_template.md](.github/pull_request_template.md) | Pull request template |
| [.github/workflows/](.github/workflows/) | Shared GitHub Actions workflows |

## Setup

To use this as your organization's profile:

1. Rename this repository to `.github` under your organization
2. Add your logo files to `assets/`:
   - `logo-light.svg` — for light mode
   - `logo-dark.svg` — for dark mode
3. Update links and content in `profile/README.md` as needed

## About

These files serve as defaults for all repositories in the Jan organization that don't have their own versions.

For more information, see [GitHub's documentation on community health files](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file).
