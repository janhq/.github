### üöÄ [Jan Server](https://github.com/janhq/server)

**Self-hostable, enterprise-grade LLM API platform**

Deploy your own AI infrastructure with a single `docker compose up`. Jan Server is a microservices platform designed for teams and organizations.

- **OpenAI-Compatible API** ‚Äî Drop-in replacement for OpenAI endpoints
- **Multi-Step Tool Orchestration** ‚Äî Response API for complex agentic workflows
- **MCP Tools Integration** ‚Äî Web search, code execution, and custom tools
- **Enterprise Auth** ‚Äî OAuth/OIDC via Keycloak with guest access support
- **Full Observability** ‚Äî OpenTelemetry, Prometheus, Jaeger, and Grafana built-in
- **Media Management** ‚Äî S3 integration for files and attachments

```bash
# Get started in minutes
git clone https://github.com/janhq/server
cd server && make quickstart
```
