# Support

## Getting Help

Here are some ways to get help with Jan:

### Documentation

- Check our [official documentation](https://jan.ai/docs)

### Community

- Join our [Discord server](https://discord.com/invite/FTk2MvZwJH) for real-time discussions
- Browse or ask questions in [GitHub Discussions](https://github.com/janhq/jan/discussions)

### Issues

- For bugs, please [open an issue](https://github.com/janhq/jan/issues/new/choose) using the appropriate template
- Search existing issues before creating a new one

### Feature Requests

- Submit feature requests through [GitHub Issues](https://github.com/janhq/jan/issues/new/choose)
- Join discussions about upcoming features in Discord

## Response Times

We do our best to respond to issues and questions in a timely manner. Response times may vary based on the complexity of the issue and current workload.
